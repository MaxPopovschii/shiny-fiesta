# Aurelia - Ristorante Italiano di Lusso

Un sito web elegante per il ristorante Aurelia, costruito con Next.js e Tailwind CSS.

## Caratteristiche

- **Homepage** con hero section e call-to-action
- **Menu Dinamico** letto da file JSON
- **Sezione Chi Siamo** con storia e foto dello chef
- **Modulo Prenotazioni** mock con toast di conferma
- Design minimalista, mobile-responsive
- Palette colori: Crema, Verde Oliva, Oro

## Come avviare

1. Installa le dipendenze: `npm install`
2. Avvia il server di sviluppo: `npm run dev`
3. Apri [http://localhost:3000](http://localhost:3000)

## Struttura

- `app/` - Pagine Next.js App Router
- `data/menu.json` - Dati del menu
- `public/` - Immagini (aggiungi hero.jpg e chef.jpg)
- `styles/globals.css` - Stili Tailwind con tema personalizzato
